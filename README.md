# E-commerce Website Design

## 📌 Project Description
This project is a basic E-commerce website that allows users to browse products, view product details, add items to a cart, and place orders online. The website demonstrates the essential structure and functionality of an online shopping platform.

## 🚀 Features
- Home page with product highlights
- Products listing page
- Product view with details
- Category-based browsing
- Add to cart functionality
- Cart and checkout system
- Order confirmation page
- About Us page
- Contact Us form with validation
- Product search feature

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- jQuery
